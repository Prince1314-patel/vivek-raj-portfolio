# Vivek Raj — Portfolio

A single-page portfolio built with Vite, React, Tailwind CSS, and Framer Motion.

## Development

```bash
npm install
npm run dev
```

## Testing

```bash
npm test
```

## Build

```bash
npm run build
npm run preview
```

## Contact form

The contact form posts to Formspree. Replace the placeholder endpoint in
`src/components/ContactForm.jsx` (`FORMSPREE_ENDPOINT`) with your real
Formspree form URL after creating a free account at https://formspree.io.
